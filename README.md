# cs2-bot

## 🧑‍💻 Kurulum
- `install-modules.bat` dosyasını çalıştırın ve modülleri indirin.
- `.env` adlı dosyayı doldurun.
- `start.bat` adlı dosyayı başlatın ve botunuz hazır!

- Özel yetkilendirme sistemi.
